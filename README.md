Overview

RawhPlayer is a lightweight desktop media player application focused on custom UI design and core playback functionality. The project demonstrates understanding of multimedia handling, GUI structuring, and user interaction flow.

Core Features

Audio playback controls (Play / Pause / Stop).

Custom UI layout.

File handling and media loading.

Interactive control elements (buttons, sliders).

Clean minimal design.

Technical Breakdown

1. Architecture

Modular code structure separating UI and logic.

Event-driven button control system.

State management for playback control.

2. Media Handling

Integration of Python multimedia libraries (e.g., pygame / similar backend).

Playback state transitions (idle → playing → paused).

Controlled resource handling.

3. UI Implementation

Custom component styling.

Interactive widgets for user input.

Organized layout structure for usability.

Skills Demonstrated

Desktop application development

GUI layout design

Event handling

Media playback integration

State management

User experience design principles

Engineering Decisions

Lightweight design over feature-heavy approach

Manual control over UI rather than relying on heavy frameworks

Emphasis on clarity and responsiveness

Potential Improvements

Playlist management

Volume and seek bar enhancement

Drag-and-drop file support

Equalizer visualization
